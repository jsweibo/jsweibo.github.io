什么是 npm 脚本

npm 脚本指的是package.json中scripts字段里的内容。

如何查看脚本

使用npm run-script命令查看package.json中的所有脚本。

注：npm run-script命令也可以缩写为npm run。

如何执行脚本

npm run-script + 脚本名称 即可执行对应的脚本。
