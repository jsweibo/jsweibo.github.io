什么是 Set

对集合的实现，是 ES2015 新增的一种引用类型。

由于集合中的元素具有唯一性，因此Set中的各个值都是独一无二的

示例：

在添加元素时，集合需要进行相等判断，这里使用的是SameValueZero算法。

即在Object.is()的基础上，把0和-0视作相等。

示例：

由于Set实现了Set.prototype[Symbol.iterator]，因此可以被for...of迭代。此外，迭代的顺序和插入的顺序是相同的。
